# Art & Audio Prompt Pack

Everything the game still wants from Gemini, in copy-paste order. Each block
is complete and self-contained — send one block per generation, in order.
When a batch comes back, upload the images in the same order they appear here
(or tell me which is which) and I'll slice + wire them.

Ground rules baked into every image prompt (don't edit them out):
- Solid flat magenta `#FF00FF` background on anything that gets chroma-keyed.
- STRICT side profile facing RIGHT for animals, one eye visible.
- No text, no watermark, no shadows, no baseline lines.
- Grids: 2×3 (six cells) for walk/run cycles, 2×2 for death/pose sets,
  NO drawn grid lines.

Status legend: every asset below already works in-game with procedural
placeholder art — painted art just drops in on top.
